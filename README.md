# Velmaya

Women's ethnic-wear D2C e-commerce platform (kurtis, kurti sets, short kurtis, co-ord sets) — Next.js 15 + Supabase + Cloudflare.

See [docs/](docs/) for brand architecture, information architecture, database schema, and policy content.

## Stack

- Next.js 15 (App Router) + TypeScript + Tailwind CSS + shadcn/ui + Framer Motion
- Supabase (Postgres, free tier)
- Cloudflare Pages (hosting) + Cloudflare R2 (image/media storage)
- Razorpay (payments)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
